Terminal Games (Snake & Tetris)

Requirements
- Linux or POSIX terminal
- Python 3
- `curses` available (standard on Linux)

Run

To run Snake:

```bash
python3 /home/daniel/Desktop/os/games/snake.py
```

To run Tetris:

```bash
python3 /home/daniel/Desktop/os/games/tetris.py
```

Launcher:

```bash
python3 /home/daniel/Desktop/os/games/launcher.py
```

Controls
- Snake: arrow keys or WASD to move, `p` pause, `q` quit
- Tetris: arrows to move/rotate/down, `q` quit

Notes
- Run in a sufficiently large terminal (at least 24x80) for best experience.
